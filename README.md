# SecLog-Analyzer-Pandas 🛡️

这是一个基于 Python 和 Pandas 开发的 **Web 安全日志自动化分析脚本**。

## 📝 项目背景
本项目是我在学习数据处理与 AI 安全开发过程中的一个练习作品。
初衷是希望将传统的“安全测评思维”与“自动化数据工程”相结合，尝试利用 Pandas 解决实际的安全日志审计痛点。

## ✨ 主要功能
- **自动化清洗**：处理原始 Web 日志中的缺失值，并进行时间序列转换。
- **攻击特征识别**：利用自定义逻辑识别常见的 SQL 注入、XSS 跨站脚本及路径遍历攻击。
- **风险画像统计**：对高风险 IP 进行多维度聚合分析（攻击频率、攻击类型多样性、响应延迟等）。
- **异常关联分析**：通过 `.query()` 快速定位疑似攻击成功的异常记录（如攻击载荷触发了 500 错误）。

## 🛠️ 技术栈
- **语言**：Python 3.10+
- **核心库**：Pandas, NumPy
- **数据导出**：OpenPyXL

## 📂 文件说明
- `data_generator.py`: 模拟生成真实的 Web 安全日志数据集。
- `analyzer.py`: 核心分析逻辑代码。
- `web_security_logs.csv`: 练习使用的原始数据集。
- `risk_report.xlsx`: 最终生成的风险分析报告。
